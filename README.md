# Markov-Mobilitas-Mahasiswa-Akhir-Sains-Data-2025
Analisis pola mobilitas harian mahasiswa Sains Data angkatan 2022 menggunakan model Rantai Markov untuk memodelkan peluang perpindahan lokasi antar empat state kampus: Kelas, Kantin, Perpustakaan, dan Kosan. Termasuk penyusunan matriks probabilitas transisi, perhitungan steady-state, visualisasi graf, dataset, dan script R.
